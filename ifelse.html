Utmaning 2 - if … else …

Datorer programmeras ofta med villkor som de kan följa.
Det är när man talar om för en dator att om det händer något speciellt så ska den göra på ett visst sätt. I programmeringsspråket python används ordet if för "om" och else för "annars".

Ett villkor kan antingen ha värdet "sant" eller "falskt".

if(pengar >=10):
print("Du får köpa chips")
else:
print("Du behöver spara pengar :)")

Skriv ett värde som variabeln "pengar" ska tilldelas så att "Du får köpa chips" skrivs ut.

Du kan behöva trycka på "spara" knappen två gånger innan svaret kollas upp.
